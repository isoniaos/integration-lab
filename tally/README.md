# Tally

This directory documents Tally compatibility notes.

Tally compatibility depends on Governor-style contracts and the event/function
surfaces that Tally supports. Treat all Tally work in this repository as a
compatibility experiment unless a production-supported integration is explicitly
scoped in product repositories.

## Current Maturity

Compatibility research note. No active IsoniaOS product surface claims Tally
integration support from this repository.

## Evidence Needed Before Integration Claim

- A verified compatible Governor-style deployment or explicit adapter model.
- Tally page/API observations for the exact contract surface being claimed.
- Source labels for which fields are Tally observations versus IsoniaOS state.
- Product-owned tests, docs, and authority-boundary review.
